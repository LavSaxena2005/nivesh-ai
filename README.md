# 📊 Nivesh AI

AI-powered stock analysis platform for Indian investors.

## Features
- Stock Analysis (RSI + Signals)
- Opportunity Radar
- Market ChatGPT
- AI Video Generator

## Tech Stack
- FastAPI
- Streamlit
- yfinance

## Run Locally

Backend:
uvicorn backend.main:app --reload

Frontend:
streamlit run streamlit_app/app.py
